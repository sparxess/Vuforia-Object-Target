Данный проект включает в себя разработку AR-Приложения 
под Android мобильное устройство на платформе Vuforia Engine 
с использованием игрового движка Unity 3D. 
Приложение построено с использованием трехмерных меток 
методом Object Target. Объектом исследования выступает устройство 
Raspberry Pi 3 Model B+ в корпусе, включая плату Sense Hat
